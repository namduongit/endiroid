**Get started**
- This repository helps everyone get started with Android programming without having to install Expo.
- How to use the project
```bash
git clone: https://github.com/namduongit/endiroid.git
cd endiroid
npm install
# To do something, edit App.js
```

**Concepts**
- React Native uses JavaScript and JSX to build mobile applications.
- Components are the building blocks of a React Native application.
- Styles are defined using JavaScript objects, similar to CSS but with some differences.  
- Navigation between screens can be handled using libraries like React Navigation.
